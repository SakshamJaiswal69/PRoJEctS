# 🤖 Jarvis AI Assistant

An intelligent desktop AI assistant built using Python that can understand voice commands, perform tasks, answer questions, and automate daily activities.

## 🚀 Features

- Voice Recognition
- Text-to-Speech
- Open Applications
- Search the Web
- AI Chat
- Persistent Memory
- Modular Architecture
- Automation Support
- Extensible Tool System

## 🛠️ Technologies

- Python
- OpenAI API
- Faster-Whisper
- SQLite
- pyttsx3 / TTS
- SpeechRecognition
- Git

## Folder Structure

```
Jarvis/
├── core/
├── voice/
├── observers/
├── automation/
├── memory/
└── main.py
```

## Installation

```bash
git clone https://github.com/SakshamJaiswal69/Jarvis.git
cd Jarvis
pip install -r requirements.txt
python main.py
```
